# 用神忌神判定

## 1. 核心原则

- 先定病，再下药。
- 用神的目标是“平衡结构、打通流通、提升可执行性”。
- 取用神不是固定口诀，必须回到当盘。
- 若采用张成达口径，先完成景象还原，再给“病药对应”。
- 若采用杨清娟盲派口径，先完成“体用与做功路径”再定用神。

## 2. 判定流程

1. 用 `strength.md` 确认日主状态与主要矛盾
2. 判断优先修复方向（扶抑、调候、通关）
3. 列出 1 到 3 个候选用神
4. 检查候选是否有根、有路、有现实承载
5. 定用神、喜神、忌神
6. 用 `luck.md` 验证岁运中是否稳定

## 3. 常见取用逻辑

- 身弱且寒：先暖后扶，避免只谈补身
- 身强且燥：先泄或克，再谈细化
- 杀重：优先考虑制杀或化杀
- 财重身弱：先保身，再谈财
- 食伤重：看是否能生财或制杀，避免空耗

## 4. 用神质量评估

高质量用神通常具备：

- 在原局有根或可被引动
- 能同时解决主要矛盾
- 岁运具备支持窗口

低质量用神常见问题：

- 理论正确但原局无根
- 一处修复引发另一处失衡
- 岁运长期不支持

## 5. 输出要求

完整输出至少包含：

1. 当前主要矛盾
2. 用神与喜神
3. 忌神及原因
4. 为什么不用备选方案
5. 哪些岁运会改变取用结论

盲派口径加一行：

6. 体用关系是否闭环（是否真正“拿到财官”）

## 6. 禁止项

- 只报“喜木火、忌金水”而不解释原因
- 不说明口径就直接给结论
- 无条件句的绝对化判断
