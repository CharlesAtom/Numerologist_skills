# 大运与流年

## 1. 目标

将“本命结构”与“时间变量”叠加，判断阶段主题、风险窗口和机会窗口。

## 2. 起运与顺逆

### 2.1 顺逆原则（常见口径）

- 阳男阴女：顺行
- 阴男阳女：逆行

### 2.2 起运岁数

- 默认采用常见起运算法（按出生时刻到最近节令换算）。
- 若用户指定软件口径，优先沿用并写明。
- 若用户指定张成达口径，按“3天1岁、1天4个月、1小时5天”折算。

## 3. 推演顺序

1. 本命结构与用忌
2. 当前大运对本命的增减作用
3. 流年叠加触发点
4. 必要时补充流月作为细化

## 4. 重点观察项

- 用神在大运是否得力
- 忌神是否被激活
- 合冲刑害是否改写主轴
- 岁运并临是否放大事件强度
- 关键十神是否进入“可兑现状态”
- 是否处于交运脱运（换运前后五年）

## 5. 常见结构信号

- 用神得运：推进效率高，成果更容易落地
- 忌神叠加：压力、纠纷、损耗上升
- 禄旺但身弱：机会多但承载力不足
- 杀旺得制：高压中有突破

## 6. 输出模板

建议按以下格式输出：

1. 当前大运主题（10年）
2. 本运最受益领域
3. 本运最高风险领域
4. 接下来 1 到 3 年关键触发点
5. 每个触发点对应的行动建议

## 7. 风险表达规范

- 不给“必然发生”的断语。
- 用“高概率窗口/需要预案”的表达。
- 对健康、法律、投资等高风险议题，必须提醒专业评估。

张成达版补充：

- 交运脱运必须单列提醒，不可省略。

杨清娟盲派补充：

- 运势判断优先看“体用做功”在该运是否可执行。
- 好运结束后可提示“余气窗口”，坏运结束后提示“惯性压力”。
