# 子平八字排盘计算参考

## 说明

本文件提供默认排盘口径与自检流程。核心原则：

- 先明确口径，再计算四柱。
- 先排盘后解读，不用结论反推盘面。
- 若用户已有固定口径，优先沿用并说明差异。

张成达口径补充：

- 若用户指定“张成达版”，起运与十二宫运行改按 `zhang-chengda.md` 执行。

## 1. 基础元素

### 1.1 十天干与十二地支

- 天干：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
- 地支：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥

### 1.2 阴阳划分

- 阳干：甲、丙、戊、庚、壬
- 阴干：乙、丁、己、辛、癸

## 2. 输入与口径确认

必须确认：

1. 出生日期（公历或农历）
2. 出生时间（最好到分钟）
3. 性别
4. 出生地与时区
5. 是否沿用某软件口径

口径争议点：

- 月柱是否以节气切换（默认是）
- 子时是否换日（需明确）
- 是否使用真太阳时（默认不使用）
- 起运算法（按用户指定或常见口径）
- 是否启用“阴阳同生同死、顺行十二宫”（张成达版）

## 3. 四柱计算顺序

1. 年柱：以立春为年界，不以春节为年界。
2. 月柱：以节令切换（节气入节时刻）定月支。
3. 日柱：按万年历或可靠算法换算。
4. 时柱：按出生时段定时支，再由日干推时干。

## 4. 月柱规则

### 4.1 月支固定对应（节令）

- 寅月：立春后
- 卯月：惊蛰后
- 辰月：清明后
- 巳月：立夏后
- 午月：芒种后
- 未月：小暑后
- 申月：立秋后
- 酉月：白露后
- 戌月：寒露后
- 亥月：立冬后
- 子月：大雪后
- 丑月：小寒后

### 4.2 月干推法（年干起月）

- 甲己年：丙寅起
- 乙庚年：戊寅起
- 丙辛年：庚寅起
- 丁壬年：壬寅起
- 戊癸年：甲寅起

之后按天干顺序排到对应月支。

## 5. 时柱规则

### 5.1 时支对照

| 时支 | 时间 |
|------|------|
| 子 | 23:00-00:59 |
| 丑 | 01:00-02:59 |
| 寅 | 03:00-04:59 |
| 卯 | 05:00-06:59 |
| 辰 | 07:00-08:59 |
| 巳 | 09:00-10:59 |
| 午 | 11:00-12:59 |
| 未 | 13:00-14:59 |
| 申 | 15:00-16:59 |
| 酉 | 17:00-18:59 |
| 戌 | 19:00-20:59 |
| 亥 | 21:00-22:59 |

### 5.2 时干推法（按日干起子时）

- 甲己日：甲子起
- 乙庚日：丙子起
- 丙辛日：戊子起
- 丁壬日：庚子起
- 戊癸日：壬子起

子时起点确定后，按天干顺序顺推到目标时支。

## 7. 输出最小结构

排盘输出至少包含：

- 四柱干支
- 各支藏干
- 十神映射（以日主为中心）
- 主要合冲刑害
- 采用口径与不确定项

## 8. 时间不确定处理

如果用户只知道模糊时段：

- 给出 1 到 2 个候选时柱。
- 明确这些候选会改变哪些结论（如子女、晚年、执行风格）。
- 不得把模糊时段当作精确时辰。

## 9. 排盘自检清单

- 是否明确采用节气换月
- 年柱是否按立春切换
- 日柱是否与可信历表一致
- 时柱是否与日干推导一致
- 藏干是否完整无误
- 十神映射是否以日主为中心
- 候选时柱差异是否给出
- 若启用张氏口径，起运折算是否按“3天1岁、1天4个月、1小时5天”
