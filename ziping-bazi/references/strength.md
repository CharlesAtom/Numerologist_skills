# 旺衰与气势判断

## 1. 目标

本文件用于判断日主状态、全局气势、调候需求。旺衰不是目的本身，而是为取用神服务。

张成达版强调：

- 先“活看景象”，再谈旺衰与格局。
- 优先找“病”（失衡点）再找“药”（用神）。

杨清娟盲派补充：

- 先定体用，再看旺衰细节。
- 旺衰不是终点，要服务于“体能否做功取用”。

## 2. 判断顺序

1. 月令司令：先看当令五行
2. 通根：看日主与关键十神在地支是否有根
3. 透干：看关键力量是否显化
4. 生克流通：看是否堵塞
5. 寒暖燥湿：看调候需求
6. 再给综合结论（偏强、偏弱、平衡、从势可能）

## 3. 通根层级（建议）

- 强根：日主在月支或多支得本气
- 中根：余气、邻支间接承托
- 弱根：仅一处浅根且受制
- 无根：地支难见同类支撑

## 4. 透干层级（建议）

- 关键十神多透且有根：可作为主轴
- 仅透无根：作用短促、易虚
- 有根不透：有潜力但不显

## 5. 调候原则（简版）

- 冬月寒重：优先考虑温暖与流通
- 夏月燥热：优先考虑润泽与降燥
- 土重壅塞：优先疏通
- 水泛木漂：优先固本定向

张成达版常用景象关键词：

- 冬木寒凝：先暖局，再论生扶
- 土重金埋：先疏土通关，再谈财官
- 水多木漂：先定根，再谈扩张

## 6. 常见误区

- 只看数量，不看月令与层级
- 先入为主判从格
- 把“身弱”机械等同“补印比”
- 忽略调候导致判断偏差

## 7. 结论输出模板

- 日主状态：偏强/偏弱/中和/从势待定
- 主要依据：月令、根气、透干、流通
- 关键矛盾：过旺、过弱、寒、燥、堵
- 后续动作：交由 `yongshen.md` 判定用忌

张成达版加一行：

- 景象病药：本局之病是___，对应之药是___
