# 合冲刑害与组合规则

## 1. 总原则

- 先确认是否成立，再判断强弱与方向。
- 单一关系不直接等于吉凶，必须回到全局与用忌。

## 2. 天干关系

### 2.1 天干五合

- 甲己合土
- 乙庚合金
- 丙辛合水
- 丁壬合木
- 戊癸合火

判断要点：

- 是“合住”、还是“合化”
- 是否具备化气条件（季节、根气、同党）
- 合后对用神是增益还是阻滞

## 4. 解释优先级

1. 月令与主轴是否被改写
2. 用神是否被冲克或被解放
3. 关键十神是否被引动
4. 是否形成持续结构而非短时扰动

## 5. 常见误判

- 见冲就断坏
- 见合就断好
- 忽略“是否化成”与“化后去向”
- 只看单点，不看岁运叠加

## 6. 输出规范

解释组合时至少写清：

- 关系是否成立
- 对哪个领域生效
- 对用神是利还是弊
- 应对建议
